## Run Setup

  ```bash
  # clone the project
  git clone git@github.com:caoxiemeihao/electron-react-vite.git

  # enter the project directory
  cd electron-react-vite

  # install dependency(Recommend use yarn)
  yarn

  # develop
  yarn dev
  ```
