## Run Setup

  ```bash
  # clone the project
  git clone git@github.com:caoxiemeihao/electron-vite-template.git

  # enter the project directory
  cd electron-vite-template

  # install dependency(Recommend use yarn)
  yarn

  # develop
  yarn dev
  ```
